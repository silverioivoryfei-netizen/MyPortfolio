# Personal Portfolio Website

## Description
A fully responsive personal portfolio website developed for academic submission. Features include orbit loading screen, moving constellation background, mouse ripple effect, glowing cursor trail, horizontal scroll skills section, timeline style education layout, interactive project cards, and floating contact elements. All pages are complete, navigation works smoothly, follows proper design standards, no overly bright colors, no broken layouts, no empty pages, and fully customized.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- GitHub Pages
